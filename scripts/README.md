# Scripts Organizer

This directory contains utility scripts organized by category:

## 📁 Directory Structure

### `/debug/`
Contains debugging and checking scripts:
- `check_*.php` - Various data validation and checking scripts
- `debug_*.php` - Debugging utilities for filters and functionality

### `/tests/`
Contains test scripts:
- `test_*.php` - Testing utilities and validation scripts

### `/data/`
Contains data creation and migration scripts:
- `create_*.php` - Data creation and seeding scripts

## 🚀 Usage

To run any script:
```bash
php scripts/[category]/script_name.php
```

## 📝 Notes

- These scripts are utility files for development and debugging
- They are not part of the main application code
- Use them for testing, debugging, and data management tasks
